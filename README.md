
# 🎬 IMDb Top 250 Movies Dashboard (Power BI)

## 📌 Project Overview

This project analyzes the **Top 250 Movies from IMDb** using Power BI.
The goal is to clean and transform the data, create meaningful measures, and build an interactive dashboard that provides valuable insights.

---

## 📂 Data Source

* Dataset obtained from **Kaggle**
* Contains information about the top 250 movies on IMDb (e.g., title, rating, year, genre, duration, certificate, etc.)

---

## 🧹 Data Cleaning & Transformation

The following steps were performed using Power Query:

* Cleaned and structured the dataset
* Created a new column **Decade** (based on the movie year)
* Created a new column **Total Duration (Minutes)** for better analysis

---

## 📊 DAX Measures

The following measures were created using DAX:

* **Number of Movies**
* **Average Duration**
* **Best Rating**
* **Lowest Rating**

---

## 📈 Key Insights

The dashboard provides the following insights:

* Number of movies by **Certificate**
* Top 5 movies by **Highest Rating**
* Number of movies by **Year**

---

## 🎛️ Interactivity

To enhance user experience, the dashboard includes slicers:

* Filter by **Year**
* Filter by **Certificate**

---

## 🚀 Tools Used

* Power BI
* Power Query (Data Cleaning & Transformation)
* DAX (Data Analysis)
